## The philosophy

No build step is not laziness — it's a design decision.

Every project loads instantly, works offline where it matters, and can be forked, saved, and hosted by anyone in under 60 seconds. Complexity is the enemy of done.

- **Single-file HTML** — one file, open in any browser, done
- **Vanilla JS** — no React, no Vue, no webpack
- **GitHub Pages** — free hosting, no server, no ops
- **Claude AI** — built collaboratively with Anthropic's Claude
